# GreenRoute AI - POC Report Template

## 1. Prototype Purpose
This prototype demonstrates how an AI-based logistics intelligence platform can help Blibli predict delivery delays, estimate shipment-level carbon emissions, calculate SLA risk, and recommend more efficient delivery routes.

## 2. Core Features
- Delivery time prediction
- SLA risk scoring
- Carbon footprint estimation
- Route recommendation
- Dashboard visualization

## 3. Demo Data
The POC uses synthetic data representing shipment distance, vehicle type, load weight, traffic condition, weather risk, hub dwell time, driver behavior score, and SLA target.

## 4. AI/Algorithm Components
- Random Forest Classifier for delay prediction demonstration
- Random Forest Regressor for carbon estimation benchmark
- Rule-based carbon calculator
- Multi-objective route scoring based on time, carbon, and SLA risk

## 5. Expected Business Impact
The solution can help logistics teams identify at-risk deliveries earlier, reduce fuel waste, improve SLA performance, and support carbon-aware operational decisions.

## 6. Limitation
The current prototype uses synthetic data. Real deployment requires integration with GPS, ERP, TMS, traffic, weather, vehicle, and hub operation data.
